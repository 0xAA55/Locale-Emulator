Introduction

Locale Emulator is a tool silmilar with MS AppLocale and NTLEA, providing a simulation function that can make an application recognize your OS as a language other than the real one. It is quite useful when you are trying to play a country-specific games.

Download

The compiled executable can be found at Github Releases page. Alternative download mirror: MEGA.

To new install

  1. Download executable from any link above.
  2. Extract all files into a SAFE folder (because you will NOT be able to move these files after installation.)
  3. Run LEInstaller.exe and press "Install / Upgrade" button.
  4. Now you will see "Locale Emulator" menu in context menu when you right clicking on files.

To update from previous version

  1. Download executable from any link above.
  2. Extract all files into the same folder and overwrite all old files.
  3. Run LEInstaller.exe and press "Install / Upgrade" button.
  4. Now you will see "Locale Emulator" menu in context menu when you right clicking on files.

System Requirement

Locale Emulator is compliable with the following OSs, both 32- and 64-bit:

  • Windows 7 with Service Pack 1
  • Windows 8 / 8.1
  • Windows 10 Build 10240 (RTM)

This tool only support 32-bit executable running in either 32-bit or 64-bit OS. That is, If you are using 64-bit WinRAR, you will not see the context menu on a .rar file, and will not be able to run WinRAR with LE.

Got any problems?

Google Chrome may block the download of Locale Emulator.

Please go to "Downloads" page (chrome://downloads/) and select "Recover file."

Some antivirus software may report LocaleEmulator.dll as a virus or block Locale Emulator from functioning.

You can add a trust item in your antivirus software, or simply remove all files of LE.

After checking antivirus software, Locale Emulator still fail.

Please make sure you have installed ALL updates in Windows Update. If problem presists, please consider to report a issue (See below).
